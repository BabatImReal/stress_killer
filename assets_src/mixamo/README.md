# Mixamo download checklist

Drop all downloaded .fbx files into THIS folder. Naming doesn't need to be perfect —
Claude will rename/convert. Target filenames and what to search on mixamo.com:

## Character (download once)
| File | Mixamo search | Pick | Download settings |
|------|---------------|------|-------------------|
| boss.fbx | Characters tab → search "suit" | any business-suit male you like as the boss | Format: **FBX Binary**, Pose: **T-Pose** |

## Animations (download each, ~12 total)
Settings for ALL: Format **FBX Binary**, Skin: **Without Skin**, FPS: 30, Keyframe Reduction: **none**.
If an "In Place" checkbox appears, TICK it (stops the animation sliding him across the floor).

| File | Search term | What to pick |
|------|-------------|--------------|
| idle.fbx | idle | a calm standing idle with subtle breathing |
| idle_angry.fbx | angry | an angry standing idle (hands on hips / fists) |
| taunt.fbx | taunt | any mocking taunt |
| point.fbx | pointing | angry pointing at camera |
| hit_head.fbx | head hit | head snapping back from a punch |
| hit_body.fbx | hit reaction | torso/gut hit reaction |
| hit_light.fbx | light hit | small flinch |
| stumble.fbx | stumble backwards | stumbling back several steps (tick In Place) |
| dizzy.fbx | dizzy | wobbly dizzy idle |
| knockdown.fbx | falling back | falling backwards to the ground (tick In Place) |
| getup.fbx | getting up | standing up from lying on back |

Optional extras if you see something funny: "shaking fist", "yelling", "brutal walk".
